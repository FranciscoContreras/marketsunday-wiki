export const frame = (tool) => [null, C, null, C, P, C, C, tool, C];

Every Work Order shares one frame: **five copper ingots around a sheet of paper**,
with the **job's tool** in the bottom-center slot. Craft the one you want, then
hand it to a copper golem — the golem takes the job on the spot.

:::note
The Work Order is used up when a golem accepts it. The recipes below are the
defaults — server owners can change shapes and costs, so check your server's
recipe book if something looks different.
:::
